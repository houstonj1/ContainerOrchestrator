WHY CONTAINER ORCHESTRATOR?

The Container Orchestrator system is a packaged Linux application that allows users to use, manage, and monitor the lifecycle of containerized applications from a web interface. The application is designed to accompany an installation of the Docker Engine, created by Docker Inc. Using the Container Orchestrator application with the Docker Engine will provide users with the ability to create and manage containers from trusted, secure Docker images within a single management interface. In addition, users who build and modify content within Docker Inc.â€™s public container registry, Docker Hub, will be able to pull, deploy and test their own applications on their own system. The Container Orchestrator application is packaged as a web application accompanied with a web server in order to host the web content and communicate with the Docker Engine. To pull images from Docker Hub, the application host is required to have an internet connection capable of reaching the Docker Hub registry.
